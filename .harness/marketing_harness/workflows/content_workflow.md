## PASO 1 — DEFINICIÓN ESTRATÉGICA Y PILARES de CONTENIDO

**Rol activo: Social Media Manager + Especialista SEO**
**Cargar: `roles/social_media_manager.md` + `roles/seo_specialist.md` + `company_context.md`**

```
1. Al inicio del mes, el Social Media Manager (SMM) define las temáticas clave que guiarán el contenido:
   - ¿Qué pilares cubriremos? (ej. 30% Educativo sobre café, 30% Inspiracional sobre coworking, 40% Venta/Promoción de eventos).
   - ¿Cuáles son los hitos o fechas especiales del mes (efemérides relevantes para la marca)?
2. Si se incluye la redacción de artículos de blog, el Especialista SEO realiza un Keyword Research y define:
   - Las palabras clave principales e intención de búsqueda.
   - El esquema H1-H3 recomendado para estructurar el artículo de manera óptima para Google.
```

**Output requerido:** Pilares temáticos validados y estructura base del calendario aprobada por el Director.

---

## PASO 2 — REDACCIÓN DE COPYS, CAPTIONS Y GUIONES

**Rol activo: Content Manager**
**Cargar: `roles/content_manager.md` + `company_context.md` + `conflict_resolution.md`**

```
1. El Content Manager toma el calendario base y escribe de forma persuasiva:
   - Las descripciones o copys de cada publicación (captions) de Instagram, LinkedIn o TikTok.
   - Los guiones detallados (Scripts) paso a paso de los Reels o videos cortos, marcando el hook y el CTA.
   - Si se planificó artículo de blog, redacta el texto completo incorporando los keywords del SEO de forma natural.
2. El Content Manager asegura que se mantenga de principio a fin la personalidad y el "Tono de Voz" de la marca.
```

**Output requerido:** Textos redactados con estructura limpia y llamadas a la acción claras.

---

## PASO 3 — PRODUCCIÓN DE ARTES VISUALES Y MULTIMEDIA

**Rol activo: Diseñador Gráfico / Multimedia**
**Cargar: `roles/designer.md` + `company_context.md`**

```
1. El Diseñador Gráfico lee las especificaciones visuales provistas por el Content Manager para cada post.
2. Diseña los banners, portadas, carruseles de fotos e infografías respetando la paleta de colores y tipografías.
3. Edita los clips de video cortos para convertirlos en Reels/TikToks premium:
   - Sincroniza audio e imágenes.
   - Coloca subtítulos con perfecta ortografía y tipografía corporativa.
   - Exporta todo el contenido visual en alta resolución y optimizado para móviles.
```

**Output requerido:** Creatividades visuales finales guardadas en las carpetas de recursos.

---

## PASO 4 — INTEGRACIÓN EN EL CALENDARIO Y REVISIÓN DE CALIDAD

**Roles activos: Social Media Manager + Todos los Especialistas**
**Cargar: `roles/social_media_manager.md` + `conflict_resolution.md`**

```
1. El Social Media Manager recopila los copys del Content Manager y las imágenes del Diseñador.
2. Integra todo de manera ordenada en el archivo de planificación: `.harness_marketing/content/content_calendar.md`.
3. Se realiza la validación de calidad grupal:
   - ¿El Diseñador aprueba que las tipografías y el uso del logo se mantengan perfectos en todas las piezas?
   - ¿El Content Manager valida que no se haya deslizado ningún error ortográfico en las composiciones visuales?
   - ¿El SEO confirma que los títulos del blog mantendrán la indexación?
```

**Output requerido:** Archivo `.harness_marketing/content/content_calendar.md` completo con copys, enlaces visuales, fecha y hora de publicación.

---

## PASO 5 — PRESENTACIÓN Y APROBACIÓN DEL USUARIO

**Rol activo: Digital Marketing Manager / Director**
**Cargar: `roles/director.md`**

```
1. El Director de Marketing presenta el calendario mensual completo e interactivo al usuario.
2. Explica qué buscamos lograr con las temáticas elegidas (ej. "este carrusel educativo busca generar guardados e interacciones, mientras que esta story busca derivar clics a WhatsApp de reservas").
3. ⛔ STOP — ESPERAR APROBACIÓN EXPLÍCITA DEL USUARIO.
4. Si el usuario pide cambios → el equipo correspondiente ajusta y actualiza la pieza.
5. Si el usuario aprueba → continuar al Paso 6.
```

---

## PASO 6 — PROGRAMACIÓN Y LANZAMIENTO

**Rol activo: Social Media Manager**
**Cargar: `roles/social_media_manager.md`**

```
1. El SMM accede a la plataforma de programación (ej. Metricool o Meta Business Suite).
2. Programa cada post en la fecha y hora de mayor actividad de la audiencia según las estadísticas de la cuenta.
3. Sube los copys exactos y las piezas visuales autorizadas.
4. Si es artículo de blog, el Especialista SEO maqueta y sube el contenido en el gestor de contenidos (CMS) de la web.
```

---

## PASO 7 — INTERACCIÓN ORGÁNICA E IMPULSO DE COMUNIDAD

**Roles activos: Community Manager + Social Media Manager**
**Cargar: `roles/community_manager.md`**

```
1. En el momento en que se publica un post, el Community Manager está atento para responder las primeras reacciones.
2. Lanza Stories interactivas para complementar el post principal y movilizar tráfico orgánico al feed.
3. Al finalizar el mes de publicaciones, el Social Media Manager analiza las métricas de Metricool/GA4 y genera el reporte consolidado `social_media_report.md`.
```

**Output requerido:** Archivo `.harness_marketing/reports/social_media_report.md` guardado con la tasa de engagement orgánica final.
